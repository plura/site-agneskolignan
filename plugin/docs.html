Collections

Collections are a category of Objects, accessed as a subitem under “Objects” in the admin sidebar.
  • Active: Indicates whether the collection is active in the database. If not active, the collection will not appear on the website.
  • Client: Indicate the clients associated with each collection. Example: For “Spring Summer 2025”, include “Fabrique” and “Goelia”. If the client is not added here, the collection will not appear on that client's page.
  • Featured Image: The main visual used to represent the collection on the website. This image will appear in collection listings and client pages.

Objects

Each object has the following fields:

  • Active: Indicates whether the object is active in the database. If not active, the object will not be loaded into the frontend.
  • Client: The client this piece was designed for.
  • Type: The category of the object, e.g., “Print”.
  • Year: The year of creation or release.
  • Materials: Materials used in the object.
  • Dimensions: The dimensions of the object.
  • Gallery Credits: Optional photographer credits for the piece.
  • Description: A textual description of the piece.
  • Gallery: Image gallery of the object.
  • Collections: Indicate the collection(s) this object belongs to. Example: “Spring Summer 2025”
  • Tags: Add relevant tags to help categorize the object. Examples: “Floral”, “Womenswear”, “Minimal”, “Gold”.
